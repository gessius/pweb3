# pweb3-pi

This is a sub module of [web3.js][repo]

This is the Eth module to be used [web3.js][repo].
Please read the [documentation][docs] for more.

## Installation

```bash
npm install pweb3-pi
```

## Usage

```js
import {Pi} from 'pweb3-pi';

const eth = new Pis(
    'http://127.0.0.1:4546',
    null,
    options
);
```
